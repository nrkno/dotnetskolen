# dotnetskolen
Workshop for oppsett av .NET-prosjekter fra bunn

## Kladd fra MOB
- `dotnet sln`
- `dotnet new console -lang F# -o src/Prosjektnavn`
- `dotnet sln add /src/Prosjektnavn/Prosjektnavn.fsproj`
- `dotnet new tool-manifest`
- `dotnet tool install paket`
- `dotnet tool restore`
- `dotnet paket init`
- `dotnet run`
